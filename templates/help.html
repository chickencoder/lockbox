Help

Uploading Files

There are two ways of selecting the file you wish to upload into a Lockbox. The first is by dragging and dropping the file from a file explorer or your Desktop into the grey area of the front page.If the drag-and-drop has been successful, then a loading animation should start. If this does not happen, try again.

The second option for selecting a file is by clicking on the grey area of the front page and manually browsing for your file through your operating system's default browser. Similarly, if successful, the area should present a loading animation to indicate that your file is being uploaded.

There are a few limitations on uploading files which could cause you upload to fail...

  1. Files must be less than 1GB is size
  2. Files should conform to standard extensions

Unlocking Files

Once a has been uploaded a file, it is important that the 4 digit pin issued by Lockbox is kept safe so that it can be easily accessed when necessary. Without this pin, uploaded files cannot be recovered.

When entering a pin to unlock a file, it is important to note that... 1. Passcodes (pins) are case sensitive 2. Passcodes are only 4 characters in length

Since uploaded files can only be accessed with a pin, Lockbox should never be used as an option for saving or backing up files. Since files are only kept for 24 hours, it is likely that if a file is stored in a Lockbox and no where else, it will be lost. Therefore, we strongly advise to only use Lockbox as a method for sharing files, not storing them.
